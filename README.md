说明
======

- `git clone https://github.com/keepeye/yii2init` 克隆到本地项目目录
- 进入目录执行 `composer install` 
- 修改配置文件，恩，主要是数据库配置
- 执行迁移 ./yii migrate
- 登录后台 /admin/index ,会自动跳转到 /admin/auth/login ，默认帐号密码 admin admin
- 查看后台功能决定是否需要二次开发
- 进行其他业务开发
